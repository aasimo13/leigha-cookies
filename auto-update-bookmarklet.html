Method 2: Manual Console Script üíª

If bookmarklets don't work, run this in the browser console:

Steps:
  1. Go to digitalcookie.girlscouts.org/scout/leigha765749
  2. Press F12 or right-click ‚Üí Inspect
  3. Click the "Console" tab
  4. Copy and paste this code:
// Extract sales data from Digital Cookie page
const goalText = document.body.innerText;
const match = goalText.match(/(\d+)\s+Packages\s+Left\s+To\s+Go/i);

if (match) {
    const packagesLeft = parseInt(match[1]);
    const goal = 300;
    const sold = goal - packagesLeft;

    const data = {
        sold: sold,
        goal: goal
    };

    localStorage.setItem('cookieSales', JSON.stringify(data));

    console.log('‚úÖ Cookie sales updated!');
    console.log('Boxes Sold:', sold);
    console.log('Goal:', goal);
    console.log('Progress:', Math.round((sold/goal)*100) + '%');
    console.log('\nNow visit leighagrace.com to see the update!');
} else {
    console.error('‚ùå Could not find sales data');
}
  1. Press Enter to run
  2. Visit leighagrace.com to see the updated count!

Method 3: Node.js Scraper Script üöÄ

For automated updates, use the scraper.js file:

Setup (one-time):
  1. Make sure Node.js is installed
  2. No additional packages needed (uses built-in modules)
Usage: 
# Just fetch and display the data
node scraper.js

# Fetch and save to admin-data.json
node scraper.js --update

Then manually update the admin dashboard with the new numbers.
